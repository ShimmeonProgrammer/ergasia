## Περιγραφή Εφαρμογής
Η εφαρμογή επιτρέπει τη διαχείριση μαθημάτων, εργασιών και βαθμολογιών για φοιτητές και καθηγητές. Οι φοιτητές μπορούν να δουν τα μαθήματα και τις εργασίες τους, να υποβάλουν εργασίες και να βλέπουν τους βαθμούς τους. Οι καθηγητές μπορούν να διαχειρίζονται μαθήματα, να αναθέτουν εργασίες και να βαθμολογούν.

## Dashboard Καθηγητή
- **RBAC έλεγχος:** Πρόσβαση μόνο σε λειτουργίες καθηγητών μέσω ελέγχου ρόλων.
- **Δυνατότητες:**
  - Δημιουργία και διαχείριση μαθημάτων (προσθήκη, επεξεργασία, διαγραφή)
  - Ανάρτηση και διαχείριση εργασιών ανά μάθημα
  - Προβολή όλων των υποβολών φοιτητών για κάθε εργασία
  - Βαθμολόγηση φοιτητών ανά εργασία και μάθημα, με δυνατότητα σχολιασμού

## Τεχνολογίες

## Οδηγίες Εγκατάστασης
1. Κάνε clone ή κατέβασε το project στον φάκελο του τοπικού server (π.χ. wamp/xampp).
2. Εισήγαγε το αρχείο `mydatabase.sql` στη βάση δεδομένων μέσω phpMyAdmin ή άλλου εργαλείου.
3. Ρύθμισε τα στοιχεία σύνδεσης στη βάση δεδομένων στα PHP αρχεία αν χρειάζεται.
4. Άνοιξε το `startpage.php` ή `login.php` μέσω του browser (π.χ. http://localhost/prototype/login.php).
5. Χρησιμοποίησε τα demo accounts ή δημιούργησε νέους χρήστες μέσω της φόρμας εγγραφής.
